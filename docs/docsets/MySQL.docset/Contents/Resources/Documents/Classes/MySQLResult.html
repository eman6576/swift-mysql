  • affectedRows

    Returns the number of rows affected by a query, if this is an Update, Insert or Delete query and this value is 0 then is can be assumed that the query has not been succesful. For Select queries the value of affectedRows will be equal to the number of rows in the dataset.

  • nextResult()

    nextResult returns the next row from the database for the executed query, when no more rows are available nextResult returns nil.

    Returns: an instance of MySQLRow which is a dictionary [field_name (String): Object], when no further rows are avaialble this method returns nil.
